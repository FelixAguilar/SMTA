Archivos SMTA: son los archivos sin configurar, para poder utilizar la
herramienta se recomienda utilizar el instalador.

Antes de instalar, se tienen que crear cuatro directorios:
    - Entrada 
    - Salida
    - Papelera
    - Instalación
No tienen que tener este nombre pero son los que se refieren en los comentarios
y en el trabajo.

SMTA: Contiene el instalador. (Ejecutar installer.sh)
